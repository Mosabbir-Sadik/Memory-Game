# Neon Memory Lab

A premium memory-match experience that pairs a tactile interface with smart stat tracking so each session feels like a personalized challenge.

## Features
- Responsive glassmorphism layout with animated aurora backdrop and purposeful typography
- Vanilla JavaScript game engine with shuffling, match validation, timer, and local best tracking
- One-time peek assist plus celebratory win modal and confetti burst
- Accessible controls (keyboard focus states, ARIA labels) and reduced-motion safeguards

## Tech stack
- Semantic HTML5 for structure and accessibility
- Modern CSS (custom properties, grid, 3D transforms) for visuals and motion
- Modular ES6 JavaScript for gameplay, storage, and UI orchestration

## Getting started
1. Download or clone the repository.
2. Open `index.html` in any modern browser. No build tools are required.
3. Flip cards to begin—the timer starts on your first move. Try to beat your saved record.

## Controls & tips
- **Shuffle deck:** Reset the board instantly via the “Shuffle deck” button or the win modal.
- **Peek assist:** Use the one-time peek per round to reveal every unmatched card for a heartbeat.
- **Goal:** Clear all pairs in the fewest moves and time. Records persist locally in `localStorage`.
